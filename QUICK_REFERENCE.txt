QUICK REFERENCE - COMMON COMMANDS
==================================

INSTALLATION & SETUP
====================

Install Dependencies:
  pip install -r requirements.txt

Verify Setup:
  python inspect_hf_dataset.py

Set Environment (Optional):
  export DEVICE=cuda
  export BATCH_SIZE=32
  export EPOCHS=50

TRAINING
========

Train with HuggingFace Dataset:
  python training/run_training.py --mode train --use-huggingface --epochs 50

Train with HuggingFace (Full Options):
  python training/run_training.py \
    --mode train \
    --use-huggingface \
    --dataset-name rokmr/cricket-shot \
    --epochs 50 \
    --batch-size 32 \
    --learning-rate 0.001 \
    --device cuda

Train with Local Data:
  python training/run_training.py --mode train --data-dir data/processed --epochs 50

Train on CPU:
  python training/run_training.py --mode train --use-huggingface --device cpu

Quick Test (5 epochs):
  python training/run_training.py --mode train --use-huggingface --epochs 5

DATASET INSPECTION
==================

Inspect Dataset (5 samples):
  python inspect_hf_dataset.py

Inspect Dataset (20 samples):
  python inspect_hf_dataset.py --samples 20

Check Label Distribution:

COMMON ISSUES & FIXES
=====================

Dataset not found:
  → Install: pip install datasets
  → Check internet connection
  → Run: huggingface-cli login

Out of memory:
  → Reduce batch size: --batch-size 16
  → Use CPU: --device cpu
  → Close other applications

Port already in use:
  → Kill existing process: lsof -ti:8000 | xargs kill -9
  → Use different port: FAST_API_PORT=9000 python backend/app.py

Module not found:
  → Install dependencies: pip install -r requirements.txt
  → Add to path: export PYTHONPATH=$PYTHONPATH:$(pwd)

GPU not detected:
  → Check: nvidia-smi
  → Use CPU: --device cpu
  → Reinstall PyTorch with CUDA support

TIPS & TRICKS
=============

1. Use --epochs 5 for testing before full training
2. Check GPU with nvidia-smi to verify device
3. Use --device cpu if testing without GPU
4. Monitor logs with tail -f for real-time feedback
5. Save checkpoints regularly to resume interrupted training
6. Use smaller batch sizes if getting OOM errors
7. Validate data with inspect_hf_dataset.py before training
8. Test API with curl before using frontend
